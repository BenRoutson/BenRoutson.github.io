MarketN


About


MarketN is an online marketplace that offers a wide array of products across grocery. With MarketN, users over Kuwait can benefit by exploring a great range of stock by various sellers and order them easily using an extremely simple user interface. It uses highly engaging aesthetics combined with an effortless process, to make grocery ordering much convenient.


What/How did?


Integrated the Stripe payment gateway that helps the users to relish hassle-free ordering experience. Introduced orders and delivery that makes it easier for the sellers to list their products. Built a system that provides greater transparency and far more efficient workflows while maintaining complete control over the internal business flows.

Technology Stack

  • Language: PHP, SQL, HTML, CSS, Javascript
  • Framework: Laravel, Vue, Bootstrap, jQuery
  • Database: MySQL
  • Payment Gateway: Stripe
  • Server: Ubuntu Apache
  • Others: Social network, Email transfer
